How to Interact with the Proof Tree

  • Collapse Nodes: Click the ⊢ symbol in any node to collapse its subtree (a button will appear in its place)
  • Expand Nodes: Click the button to expand collapsed nodes and show hidden subtrees
  • Toggle Technical Details: Use "show bunch" buttons to reveal the bunch and "show test level" buttons to display test level
  • Navigate Layout: Scroll horizontally to view different parts of the proof tree (➔)

Tip: Start with top-level nodes and collapse/expand sections to manage complexity. Blue buttons always indicate interactive elements.
